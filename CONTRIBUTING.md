# Contributing to std

Thank you for your interest in contributing!

## Workflow
1. Fork the repository.
2. Create a feature branch.
3. Commit your changes.
4. Open a Pull Request.

## Coding Style
Follow the SageLang style guide as defined in the main [SageLang repository](https://github.com/sagelang/sagelang).

## Testing
Please run the tests located in the `tests/` directory:
```bash
make test
```
Ensure all tests pass before submitting your PR.
